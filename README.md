# Canton Transparency Working Group (CTWG)

## Overview

The Canton Transparency Working Group (CTWG) is a public-good contributor initiative focused on enhancing governance clarity, Development Fund transparency, and standardized ecosystem reporting for the Canton Network.

The initiative operates independently while coordinating with the Canton Foundation under the governance framework defined in CIP-0082 and CIP-0100.

---

## Mission

As Canton adoption scales, governance visibility and milestone accountability become increasingly important.

CTWG aims to establish a structured transparency infrastructure layer that supports:

- Governance clarity
- Development Fund visibility
- Milestone tracking standardization
- Ecosystem-wide reporting consistency

---

## Scope of Work

The CTWG initiative includes:

### 1. Governance Mapping Framework
- Structured documentation of CIP-0082 and CIP-0100
- Governance lifecycle visualization
- Fund allocation model explanation

### 2. Development Fund Allocation Model
- Programmatic 5% emission allocation overview
- Sustainability characteristics
- Governance oversight mapping

### 3. Grant Transparency Dashboard (MVP)
- Governance overview interface
- Lifecycle tracking model
- Allocation structure documentation
- Reporting archive

### 4. Quarterly Transparency Reporting
- Governance activity summaries
- Proposal tracking overview
- Milestone delivery status
- Ecosystem risk and outlook analysis
